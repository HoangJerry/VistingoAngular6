### About
Angular 6 mini test Vistingo
### Prerequisites

You need to install:
```
npm
node
```

### Installing

In VistingoTest directory

run install node_module

```
npm install
```

### Running the tests
* Run and check on localhost:4200
```
ng serve
```

or
* for auto open your browser
```
ng serve -o
```

* build to js file
```
ng build
```
